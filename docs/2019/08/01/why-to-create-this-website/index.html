一天在Youtube上闲逛，看到了这位“火锅大王”的视频：


(点击可以播放视频)

如果有人喜欢德国、美国或其他任何地方，没问题。
但如果有人喜欢中国，那就不可能是真的。

醍醐灌顶！

这个网站想告诉大家，”爱中国“，不仅可以，不仅不丢人，而且是光荣的事儿！
